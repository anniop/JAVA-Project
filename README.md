Command Line ChatApp using Java Socket Programming
Overview
This project is a simple command-line based chatting application built using Java socket programming. It consists of two main components: Client.java and Server.java. The application allows clients to connect to a central server and communicate with each other in real-time, irrespective of whether they are on the same network or not.

Features
Real-time Communication: Clients can send and receive messages instantly through the server.
Cross-Network Communication: Clients can connect and chat with each other even if they are not on the same network.
Command-Line Interface: The application operates entirely through the command line, making it lightweight and easy to use.
Scalability: The architecture allows for scalability, accommodating a large number of clients simultaneously.
Getting Started
To use this application, follow these steps:

Clone the repository to your local machine.
Compile both Client.java and Server.java files.
Run the Server.java file to start the server.
Run multiple instances of Client.java to simulate multiple users.
Connect the clients to the server using the specified IP address and port number.
Start chatting using the command-line interface!
Requirements
Java Development Kit (JDK)
Any Java IDE or compiler (e.g., IntelliJ IDEA, Eclipse)
